Golang xtask demo
=====================

Use xtask for your Go project scripts.

```shell
go run xtask/main.go hello
```

# Standard tasks

* `--help`: list all available tasks, and format is `name - description`

# Tools & Libraries

* Command Line - Awesome Go: https://awesome-go.com/command-line/
